## 🟡 Warnings (Nên sửa)
1. **Rớt Unit Tests kẹt vòng Build TypeScript**
   - **File:** `src/app/api/market-data/route.test.ts` và `src/lib/market-data/crawler.test.ts`
   - **Mô tả:** Hệ thống MarketSnapshot vừa được nâng cấp tính năng mới (chèn `silver`, `news` và `aiSummary`) thế nhưng các kịch bản test cũ chưa ăn khớp. (Lỗi `TS2739: missing properties`).
   - **Nguy hiểm:** Vercel hoặc CI/CD pipeline thường sẽ chặn đứng ngay vòng kiểm thử/build nếu phát hiện TypeScript báo lỗi thế này. App sẽ không deploy được bản mới trừ khi bypass.
   - **Cách sửa:** Sửa lại các mock data object trong test file để thêm các property rỗng/mặc định hợp chuẩn type.

2. **Rủi ro đứt gãy dữ liệu hàng loạt từ Third-Party**
   - **File:** `src/lib/market-data/stock-screener.ts`
   - **Mô tả:** Vụ sập API TCBS vừa rồi là triệu chứng rõ nhất. Hệ thống frontend Next.js gọi API nhưng API gốc từ bên thứ ba (third-party) trả về `404` hay Timeout.
   - **Nguy hiểm:** Người dùng sẽ thấy loading ảo tưởng mòn mỏi hoặc một con số mốc nếu rơi vào kịch bản Fetching lâu gây tắc nghẽn server Nextjs API routing.
   - **Cách sửa:** Mọi API ngoại tuyến cần dùng Error Boundary, timeout threshold (VD ngắt kết nối sau 3000ms) kết hợp Redis/In-memory Cache làm màn chắn vĩnh cửu.
